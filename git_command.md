> 파일 생성
```$ touch 파일명
   $ cat > 파일명
   $ vi 파일명

```

> 파일 삭제
``` 
   $ rm -rf 파일명
```

> 디렉토리 삭제<br>

``` $ rmdir 디렉토리명
```

> 새로운 local repository 생성<br>

``` $ git init ```

> add한 파일을 local repository에 저장
<br>
``` $ git commit```

> local repository를 remote repository에 업로드<br>

``` $ git push ```
